# v2.3.1 - Code Cleanup & Material UX

**Release Date:** 2026-04-13

## Improvements
- **Material Search**: Multi-term search support (e.g. "Eryone PLA" filters by both brand and type)
- **Material Filters**: Filter by type (dropdown), brand (combobox) and color (search) on the materials page
- **Material Favorites**: Star materials to mark them as favorites, filter by favorites, favorites shown in material picker
- **Material Picker**: Favorites displayed at the top when opening the picker on the add spool page
- **Server-side Pagination**: Materials list loads instantly with paginated server queries instead of loading all materials at once
- **Material Cache**: Materials are cached on app start for faster navigation
- **Spool Form**: Added "Location" header above printer/storage/dryer dropdowns
- **Quick Start**: Removed materials step (already pre-loaded)

## Code Quality
- Extracted all 46 Blazor components to code-behind files (.razor.cs)
- Fixed ~180 inline if statements across the codebase
- Replaced 4-tuple with named record (OpenSpoolParseResult)
- Split ApiServices.cs into 6 individual service files
- Increased material search limit to 250
