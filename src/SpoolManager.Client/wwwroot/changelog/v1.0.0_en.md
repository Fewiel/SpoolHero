# v1.0.0 - Initial Release

**Release Date:** 2026-03-15

## Features
- Spool tracking with weight, color, material, and location
- Material database with import/export (Base64)
- NFC tag support via OpenSpool standard (NTAG215/216)
- Multi-user projects with invitation codes
- Dryer monitoring with timer and notifications
- Inventory scanning via NFC
- Printer, storage location, and dryer management with images
- Admin panel with user management, SMTP, branding, and legal settings
- Email notifications for spool levels, dryer cycles, and ticket replies
- Support ticket system
- Dark mode and language switching (DE/EN)
- Mobile-friendly responsive design
