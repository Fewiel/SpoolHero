# v2.4.0 - NFC Tag Downloads

**Veroeffentlicht:** 13.04.2026

## Neue Features
- **NFC-Tags herunterladen**: Tag-Daten als .bin (NDEF-Binaer) oder .json (Text) herunterladen zum manuellen Schreiben mit externen NFC-Tools
- **Tag-Downloads fuer alle Entitaeten**: Download-Buttons bei Druckern, Lagerorten und Trocknern (nicht nur Spools)
- **JSON anzeigen**: NFC-Tag-JSON direkt in der App anzeigen mit Kopieren-Button
- **Authentifizierte Downloads**: Tag-Downloads funktionieren jetzt korrekt mit Authentifizierung (keine 401-Fehler mehr)

## Verbesserungen
- Dark-Mode-Unterstuetzung fuer JSON-Anzeige (korrekte Text- und Hintergrundfarben)
- Mobilfreundliche JSON-Anzeige mit Zeilenumbruch
- Kopieren-Button als Overlay auf der JSON-Vorschau fuer schnellen Zugriff
