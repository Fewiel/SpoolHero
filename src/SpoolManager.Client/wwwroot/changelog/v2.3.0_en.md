# v2.3.0 - Forgot Password & Resend Verification

**Release Date:** 2026-04-12

## New Features
- **Forgot Password**: Reset your password via email link from the login page
- **Reset Password Page**: Set a new password using the emailed token (valid 1 hour)
- **Resend Verification Email**: Resend activation link on the verification page when the token expired

## Security
- Rate limiting on forgot password requests per IP (5 attempts, then 15 min lockout)
- Rate limiting on reset password attempts per IP to prevent token brute-force
- Email cooldown: max 1 reset email per 2 minutes per account to prevent email bombing
- Resend verification cooldown to prevent spam
- Password reset invalidates all existing sessions (TokenVersion increment)
- No user enumeration: forgot password always returns the same response
