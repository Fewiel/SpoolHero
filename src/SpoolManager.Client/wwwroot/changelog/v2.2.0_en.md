# v2.2.0 - Hotfixes & Changelog

**Release Date:** 2026-04-11

## Fixes
- **Unicode/Emoji Support**: All database tables migrated to utf8mb4 - emojis now work everywhere
- **PNG Transparency**: Uploaded PNG images now preserve transparency instead of black background
- **Project Creation**: Creating a second project is now possible
- **Page Headers**: Removed focus outline on page titles
- **Scrollbar**: Fixed unnecessary scrollbar on every page caused by footer
- **Cancel Button**: Disabled on project creation when no project exists yet
- **Placeholder Translation**: Project form placeholders now follow the selected language
- **Ticket Form**: New Ticket button hides when form is open instead of toggling
- **Project Name Validation**: Maximum 200 characters enforced on client and server
- **Ticket Description**: Maximum 5,000 characters enforced

## New Features
- **Changelog Page**: View all release notes directly in the app
