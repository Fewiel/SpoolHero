# v2.0.0 - Open Filament Database & Orca Slicer Export

**Release Date:** 2026-04-10

## New Features
- **Open Filament Database**: Sync 14,000+ filament materials from the community-maintained OFD via admin panel
- **Orca Slicer Export**: Export filament profiles as Orca/Snapmaker Orca compatible JSON files
- **Community Suggestions**: Users can submit material improvements; admins review and approve
- **Material Picker**: Searchable dropdown component with color swatch, brand, type, and temperature
- **Admin Suggestions Page**: Review, approve, or reject community material submissions

## Improvements
- Client-side search and pagination (50 items per page) for material lists
- Material lists load instantly after initial fetch
- New navigation entry for Orca Export
- Temperature warning when exporting materials with missing data (0 degC)
