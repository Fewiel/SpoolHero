# v1.0.0 - Erstveroeffentlichung

**Veroeffentlicht:** 15.03.2026

## Features
- Spulenverwaltung mit Gewicht, Farbe, Material und Standort
- Materialdatenbank mit Import/Export (Base64)
- NFC-Tag-Unterstuetzung via OpenSpool-Standard (NTAG215/216)
- Mehrbenutzer-Projekte mit Einladungscodes
- Trocknungsueberwachung mit Timer und Benachrichtigungen
- Inventur-Scanning via NFC
- Drucker-, Lagerort- und Trocknerverwaltung mit Bildern
- Admin-Panel mit Benutzerverwaltung, SMTP, Branding und Rechtstexten
- E-Mail-Benachrichtigungen fuer Spulenstand, Trockner und Tickets
- Support-Ticket-System
- Dark Mode und Sprachwechsel (DE/EN)
- Mobil-optimiertes responsives Design
