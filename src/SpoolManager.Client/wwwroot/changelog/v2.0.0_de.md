# v2.0.0 - Open Filament Database & Orca Slicer Export

**Veroeffentlicht:** 10.04.2026

## Neue Features
- **Open Filament Database**: Ueber 14.000 Filament-Materialien aus der Community-gepflegten OFD synchronisieren
- **Orca Slicer Export**: Filament-Profile als Orca/Snapmaker Orca kompatible JSON-Dateien exportieren
- **Community Vorschlaege**: Nutzer koennen Materialverbesserungen einreichen, Admins pruefen und genehmigen
- **Material Picker**: Durchsuchbares Dropdown mit Farbvorschau, Marke, Typ und Temperatur
- **Admin Vorschlaege-Seite**: Community-Vorschlaege pruefen, genehmigen oder ablehnen

## Verbesserungen
- Clientseitige Suche und Pagination (50 Eintraege pro Seite) fuer Materiallisten
- Materiallisten laden nach erstem Abruf sofort
- Neuer Navigationseintrag fuer Orca Export
- Temperaturwarnung beim Export von Materialien mit fehlenden Daten (0 Grad)
