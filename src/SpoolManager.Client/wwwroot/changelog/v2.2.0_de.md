# v2.2.0 - Hotfixes & Changelog

**Veroeffentlicht:** 11.04.2026

## Fehlerbehebungen
- **Unicode/Emoji Support**: Alle Datenbanktabellen auf utf8mb4 migriert - Emojis funktionieren jetzt ueberall
- **PNG Transparenz**: Hochgeladene PNG-Bilder behalten jetzt ihre Transparenz statt schwarzem Hintergrund
- **Projekt erstellen**: Ein zweites Projekt kann jetzt angelegt werden
- **Seitentitel**: Fokus-Rahmen um Seitenueberschriften entfernt
- **Scrollbar**: Unnoetiger Scrollbalken auf jeder Seite durch Footer behoben
- **Abbrechen-Button**: Deaktiviert bei Projekterstellung wenn noch kein Projekt existiert
- **Platzhalter-Uebersetzung**: Projektformular-Platzhalter folgen jetzt der gewaehlten Sprache
- **Ticket-Formular**: Neues-Ticket-Button wird ausgeblendet wenn Formular geoeffnet statt zu togglen
- **Projektname-Validierung**: Maximal 200 Zeichen auf Client und Server erzwungen
- **Ticket-Beschreibung**: Maximal 5.000 Zeichen erzwungen

## Neue Features
- **Changelog-Seite**: Alle Release-Notes direkt in der App einsehbar
