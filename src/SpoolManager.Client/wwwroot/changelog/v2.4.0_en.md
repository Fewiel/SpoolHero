# v2.4.0 - NFC Tag Downloads

**Release Date:** 2026-04-13

## New Features
- **Download NFC Tags**: Download tag data as .bin (NDEF binary) or .json (text) for manual writing with external NFC tools
- **Tag Downloads for All Entities**: Download buttons available on printers, storage locations and dryers (not just spools)
- **Show JSON**: View the NFC tag JSON payload directly in the app with a copy-to-clipboard button
- **Authenticated Downloads**: Tag downloads now work correctly with authentication (no more 401 errors)

## Improvements
- Dark mode support for JSON display (proper text and background colors)
- Mobile-friendly JSON display with word-wrap
- Copy button overlay on JSON preview for quick clipboard access
