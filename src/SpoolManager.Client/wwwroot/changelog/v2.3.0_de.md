# v2.3.0 - Passwort vergessen & Aktivierungslink erneut senden

**Veroeffentlicht:** 12.04.2026

## Neue Features
- **Passwort vergessen**: Passwort per E-Mail-Link von der Login-Seite zuruecksetzen
- **Passwort-Reset-Seite**: Neues Passwort setzen ueber den per E-Mail erhaltenen Link (1 Stunde gueltig)
- **Aktivierungslink erneut senden**: Neuen Aktivierungslink auf der Verifizierungsseite anfordern wenn der Token abgelaufen ist

## Sicherheit
- Rate-Limiting fuer Passwort-Reset-Anfragen pro IP (5 Versuche, dann 15 Min Sperre)
- Rate-Limiting fuer Reset-Token-Versuche pro IP gegen Brute-Force
- E-Mail-Cooldown: maximal 1 Reset-E-Mail pro 2 Minuten pro Konto gegen E-Mail-Bombing
- Cooldown beim erneuten Senden des Aktivierungslinks gegen Spam
- Passwort-Reset macht alle bestehenden Sitzungen ungueltig
- Keine Benutzer-Aufzaehlung: Passwort-Vergessen gibt immer die gleiche Antwort
